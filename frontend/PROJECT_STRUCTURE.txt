DIRECTORY PURPOSES
==================

public/
  Purpose: Static assets that are served directly without processing
  Contents: Favicon, images, fonts (if added later)
  Access: Available at root URL (e.g., /vite.svg)

src/components/
  Purpose: Reusable React components
  Status: Empty, ready for next phase
  Planned: InputForm.jsx, ResultsDisplay.jsx, Visualization.jsx

src/services/
  Purpose: API clients, data services, utility functions
  Contents: apiClient.js (Axios-based API communication)
  Planned: Additional services as needed

KEY FILES
=========

1. package.json
   - Defines all project dependencies
   - Contains npm scripts (dev, build, preview)
   - Project metadata

2. vite.config.js
   - Vite build configuration
   - Dev server settings (port 3000)
   - API proxy configuration (proxies /api/* to http://localhost:5000)

3. src/main.jsx
   - Application entry point
   - Renders App component into DOM
   - Enables React Strict Mode

4. src/App.jsx
   - Root React component
   - Contains header, main content area, and footer
   - Placeholder UI for next phase

5. src/services/apiClient.js
   - Exports calculateRetirement() function
   - Exports checkApiHealth() function
   - Handles API errors

DEPENDENCIES
============

Production:
  - react@^18.3.1               Core React library
  - react-dom@^18.3.1           React DOM rendering
  - axios@^1.7.9                HTTP client for API calls
  - recharts@^2.15.0            Charting library

Development:
  - vite@^6.0.5                 Build tool and dev server
  - @vitejs/plugin-react@^4.3.4 React support for Vite

NPM SCRIPTS
===========

npm run dev
  - Starts development server on port 3000
  - Enables hot module replacement
  - Proxies /api/* to http://localhost:5000

npm run build
  - Creates optimized production build
  - Output directory: dist/

npm run preview
  - Previews production build locally
